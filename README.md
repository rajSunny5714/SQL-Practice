## 📝 Solution Format

Each solution includes:

- Problem Name
- Platform
- Difficulty
- SQL Query
- Short Explanation (where applicable)

Example:

```sql
-- Problem: Second Highest Salary
-- Platform: LeetCode
-- Difficulty: Medium

SELECT MAX(salary) AS SecondHighestSalary
FROM Employee
WHERE salary < (
    SELECT MAX(salary)
    FROM Employee
);
```
